# Assignment 2 — Single-Table Patient Roster (SQLite)

This project demonstrates a simple two-step SQLite workflow:

1. Create a database from a SQL schema using Python
2. Load CSV data using Pandas and SQLAlchemy

## Project Structure
- `sql/schema.sql` — table definition
- `data/patients.csv` — source data
- `src/create_db.py` — creates the SQLite database
- `src/import_csv.py` — imports CSV data
- `sql/analysis.sql` — example single-table queries
